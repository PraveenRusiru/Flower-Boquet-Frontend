- **[https://flower-boquet-frontend.vercel.app**](https://flower-boquet-frontend.vercel.app/)

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- React Router
- Axios / Fetch

## Main Features

- Product listing (bouquets / pots)
- Admin dashboard UI
- Product create/update with image upload (up to 5 images)
- Orders and customers views (if implemented)
- Responsive layout
